**AnimeEngine** é uma plataforma robusta para gerenciamento e consumo de animes e mídias, focada em performance moderna, gamificação e segurança institucional.

---

## 📂 Organização do Projeto

O repositório foi organizado para preservar o histórico de evolução enquanto foca no desenvolvimento das versões futuras:

- **`sitev7/`**: Versão oficial estável. Backend PHP, sistema de perfil, conquistas e segurança ativa.
- **`versoesPassadas/`**: Arquivo histórico contendo as builds da v1 até a v6.
- **`testes/`**: Área experimental para subprojetos como *MangaEngine*, minijogos e protótipos.
- **`planov8.md`**: Roadmap detalhado para a próxima revolução do sistema (Arquitetura SPA).

---

## ✨ Recursos da Versão Atual (v7)

- 🛡️ **Segurança Rigorosa:** Autenticação via `mysqli` com prepared statements e controle de acesso modular.
- 🏆 **Gamificação:** Sistema de conquistas unificado entre PHP e JS, com salvamento persistente no banco de dados.
- 📊 **Stats e Streaks:** Acompanhamento dinâmico de progresso, horas assistidas e nível do usuário.
- ✨ **Design Premium:** Interface baseada em Neo-Brutalist design com variáveis dinâmicas de tema.

---

## 🛠️ Tecnologias

- **Backend:** PHP 8.x Nativo (Driver `mysqli`).
- **Frontend:** Vanilla JavaScript (ES6+), Interações Assíncronas (Fetch API).
- **Estilos:** CSS3 Modular com variáveis globais e temas dinâmicos.
- **Banco de Dados:** MySQL/MariaDB.

---

## 🚀 Como Executar

### Pré-requisitos
- Servidor local (XAMPP, WAMP ou Docker).
- PHP 7.4 ou superior.
- MySQL/MariaDB.

### Instalação
1. Clone o repositório na pasta `htdocs` do seu servidor.
2. Importe o banco de dados (script localizado em `sitev7/database/`).
3. Configure as credenciais no arquivo `sitev7/includes/database.php`.
4. Acesse `http://localhost/seu-diretorio/index.html` para ver o seletor de versões.
